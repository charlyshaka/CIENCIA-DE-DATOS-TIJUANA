#R Programing (Programación en R)
=========================
 Este repositorio va dirigido a las personas hispanas que están cursando la Especialización
de "Ciencia de Datos" en Coursera ofrecida por "Johns Hopkins University".

En este curso aprenderás cómo programar en R y cómo usar R para un análisis de datos eficaz.  Aprenderás cómo instalar y configurar el software necesario para un entorno de programación estadística y a describir conceptos genéricos de lenguaje de programación, a medida que se aplican en un lenguaje estadístico de alto nivel. El curso abarca aspectos prácticos de estadística informática que incluyen programación en R, lectura de datos en R, acceso a paquetes R, escribir funciones R, depurar, caracterización de código R y organización y anotación de código R. Los temas relacionados con el análisis estadístico de datos aportarán ejemplos prácticos.
